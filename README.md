# Data Science for Chemistry and Engineering (CHEM 452 / CHBE 413)

This repository contains the GitHub Pages / Jekyll site for the course
**Data Science for Chemistry and Engineering** taught by Prof. Nick Jackson.

The site is configured as a *project site* hosted at:

> https://thejacksonlab.github.io/ds-chem-eng

To run locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000/ds-chem-eng` in your browser.
